La LibLapin neccessite plusieurs bibliothèques pour fonctionner:

  • La bibliothèque SFML2
  • La bibliothèque PosixThread
  • La bibliothèque OpenCV (optionnelle, requise pour bunny_camera)
  • La bibliothèque LibUsb (optionnelle, requise pour bunny_hardware)
  • La bibliothèque MySQL (optionnelle, requise pour bunny_database)
Ainsi que toute dépendances dont dépendraient ces dépendances.
